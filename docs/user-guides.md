# User guides

## Administrator

Use the dashboard for organization metrics, provision users with the minimum required role, manage course publication, review reports, and revoke invalid certificates. Do not share initial passwords through public channels. Deactivating an account revokes its active sessions.

## Instructor

Create and update only owned courses, organize sections and lessons, publish assessments after verifying questions, review learner attempts, and issue certificates only after the server reports eligibility. Correct answers are never visible to learners before submission.

## Department manager

The dashboard and reports are restricted to employees whose department matches the manager profile. Review completion, assessment pass rates, active learners, and certificates. If no department is assigned, scoped data access is denied.

## Learner

Enroll in available courses, complete lessons deliberately, take final assessments within the displayed time limit, use the library, and retrieve active certificates. Video lessons require sufficient playback progress before completion. Use the public verification screen to validate a certificate code.

## Super administrator

Reserve this role for system bootstrap, security settings, and exceptional operations. Use an administrator role for normal daily management.
